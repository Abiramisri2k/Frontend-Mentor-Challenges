- Improved understanding of **Flexbox layouts** for two-column and stacked mobile designs
- How to center layouts using **Flexbox instead of absolute positioning**
- Writing **clean and semantic HTML** using `section`, `main`, and lists
- Using **CSS variables** for consistent theming
